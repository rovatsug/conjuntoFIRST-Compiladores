Ferramenta construída para auxiliar o estudo da disciplina de Compiladores em Engenharia de Computação, 
Ciência da Computação e áreas correlatas. Determinando o conjunto FIRST dada uma Gramática Livre de 
Contexto inserida corretamente, muito útil na análise e construção da análise sintática (parser) de 
uma nova linguagem de programação, independente do método selecionado.

Os formatos de entrada padrão vigentes (das gramáticas) podem ser conferidos nos arquivos de texto 
inDIGIT. Nas entradas, letras maíúsculas representam variáveis não terminais e minúsculas as terminais.

A primeira versão funcionava corretamente e compreendia apenas variáveis (ambos os tipos) de um unico
caractere. Já a segunda, está sendo construída para compreender variáveis mais longas.
Isso sem realizar nenhum tipo de gambiarra no código através de um dicionário, por exemplo.

Para um bom uso da nova versão, escreva VARIÁVEIS NÃO TERMINAIS COM TODAS AS LETRAS MAIÚSCULAS e variáveis
terminais, o contrário.
